# 样式文件目录（qcss）

本目录存放 Qt 样式表（QSS）文件，用于定义应用程序的视觉外观。

---

## 文件说明

### `style.qcss`：主样式表文件，定义了应用程序的默认视觉风格。

该文件使用 Qt 属性选择器（如 `[type="..."]` 和 `[mode="..."]`）来区分不同控件，目前支持浅色模式（`mode="light"`）。具体样式规则如下：

#### 主页面背景
- **选择器**：`QWidget[type="main_page"][mode="light"]`
- **样式**：线性渐变背景，从左上方（浅粉色）过渡到右下方（浅蓝色）。
  ```css
  background: qlineargradient(x1: 0, y1: 0, x2: 1, y2: 1,
                              stop: 0 rgba(255, 182, 193, 1),
                              stop: 1 rgba(135, 206, 250, 1));
  ```

#### 左侧按钮栏
- **选择器**：`QWidget[type="btns_bar"][mode="light"]`
- **样式**：半透明白色背景（透明度 0.3），圆角 8 像素。
  ```css
  background-color: rgba(255, 255, 255, 0.3);
  border-radius: 8px;
  ```

#### 普通按钮
- **选择器**：`QPushButton[type="btns"][mode="light"]`
- **基础样式**：
  - 背景：半透明白色（透明度 0.6）
  - 边框：2 像素实线白色
  - 内边距：上下 10 像素，左右 24 像素
  - 圆角：6 像素
  - 字体：16 像素，加粗
  - 文字颜色：黄色 (`rgba(255, 255, 0, 1)`)
- **悬停状态**（`:hover`）：背景变为半透明浅蓝色。
- **按下状态**（`:pressed`）：背景变为半透明皇家蓝。

#### 主图片标签
- **选择器**：`QLabel[type="main_image_label"][mode="light"]`
- **样式**：
  - 背景：半透明白色（透明度 0.6）
  - 文字居中，16 像素加粗，黄色
  - 圆角：6 像素

所有样式均基于 `type` 和 `mode` 属性进行匹配，便于后续扩展暗色模式或其他主题。
