# 行为准则

## 我们的承诺

为了营造一个开放、友好的环境，我们作为贡献者和维护者承诺：无论年龄、体型、身体条件、种族、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、外貌、宗教信仰、性取向等，参与我们的项目和社区都将是一种无骚扰的体验。

## 我们的标准

有助于创造积极环境的行为包括：

- 使用友好和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表现出同理心

不可接受的行为包括：

- 使用性暗示、性化语言或图像，以及不受欢迎的性关注或挑逗
- 发表挑衅、侮辱/贬损评论，以及进行人身或政治攻击
- 公开或私下骚扰
- 未经明确许可，发布他人的私人信息（如住址、电子邮件）
- 在专业环境中可能被合理认为不适当的其他行为

## 我们的责任

项目维护者有责任明确可接受行为的标准，并对不可接受的行为采取适当、公平的纠正措施。

项目维护者有权删除、编辑或拒绝不符合本行为准则的评论、提交、代码、编辑、Issue 及其他贡献，并在适当时说明审核决定的原因。

## 适用范围

本行为准则适用于所有项目空间（如 Issue、PR、讨论区），也适用于个人在公共场合代表项目或其社区时（如使用官方邮箱、通过官方社交媒体账号发言等）。

## 执行

如有辱骂、骚扰或其他不可接受的行为，可以通过项目作者的联系方式（见 README）向项目团队报告。所有投诉都将被及时审查和调查，并得到必要且适当的回应。项目团队有义务对事件报告者保密。

## 执行指南

项目维护者将遵循以下社区影响指南来确定违反本行为准则的后果：

### 1. 纠正

**社区影响**：使用不恰当的语言或其他在社区中被认为不专业或不受欢迎的行为。

**处理**：由维护者私下发出书面警告，说明违规的性质，并解释为何行为不当。可要求公开道歉。

### 2. 警告

**社区影响**：通过单个事件或一系列行为违规。

**处理**：警告并明确后续违规将导致临时封禁或永久封禁。在指定时间内，不得与相关人员进行任何形式的互动（包括私下）。

### 3. 临时封禁

**社区影响**：严重违反社区标准，包括持续的不当行为。

**处理**：指定时间内禁止与社区进行任何形式的公开或私下互动。在此期间，任何导致封禁延长的互动行为都可能导致永久封禁。

### 4. 永久封禁

**社区影响**：表现出违反社区标准的模式，包括持续的骚扰、攻击或贬低他人。

**处理**：永久禁止在社区内进行任何形式的公开互动。
